You can exit the game by pressing the X button in the top right corner of the window, like any other Windows application. Move the mouse to the top right corner of the screen and wait a second until the application bar appears, then tap on the X button.

By default the game runs in fullscreen. To make the game run in window mode go to Settings -> Exit Fullscreen.

If you wish to enter Fullscreen mode again follow the same steps: go to Settings -> Enter Fullscreen.
