Unlock Puzzles with Crowns

Author: Mihai, last modified: 17/09/2024


Step by step guide

To unlock puzzle packs with crowns, you need to collect enough crowns to unlock a puzzle pack. Read Game Economy for more details.

Tap on Unlock with crowns

Pick the Unlock with crowns button.

Confirm the Unlock

Confirm the unlock.

Start Playing

The pack is now yours, start playing the puzzle pack.

The pack is added to your library

You can now play any puzzle in the new puzzle pack that you own.
